## Spring Boot Example

This is a basic example of how to use Kotlin in a Spring Boot application. See the [accompanying tutorial](http://kotlinlang.org/docs/tutorials/spring-boot-restful.html)
for more information.

To run:

```
$ gradle bootRun
```


## features

* restful 风格api举例
* Token 身份验证,使用过滤器
* 方法级别权限控制api接口
* 处理异常自定返回结果 
* servlet异步响应
* model数据处理
